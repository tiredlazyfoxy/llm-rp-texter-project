(`WorldLocation`, `WorldNPC`, `WorldLoreFact`) before insert. The check
belongs in the `db/` layer (a single helper such as
`db.worlds.document_id_exists(id: int) -> bool`, naming up to the coder) or
inlined into the existing `db.worlds` create helpers — but **not** in
`services/` or `routes/` directly. Layer separation rule is firm.
