# Project Memory

## Key Decisions
- LLM-driven RPG chat app: world/NPCs defined, actions/dialogue generated by LLM
- "MCP tools" = OpenAI-compatible function calling via PythonLLMClient (not actual MCP protocol)
- All tool implementations: internal async functions only (no bash, no external HTTP)
- Strict typing: Pydantic for API/tools/DB, TypedDict for internal, no free dicts
- ORM: SQLModel (SQLAlchemy + Pydantic)
- LLM client: PythonLLMClient (git+https://github.com/Iezious/PythonLLMClient.git)
- Auth: JWT tokens, shared login between User and Admin SPAs
- Frontend: Vite multi-page app (same port 8094), separate entry points for user and admin
