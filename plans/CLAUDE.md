# Plans Folder

Working area for planning documents, task breakdowns, and drafts.

## Rules
- This folder is **git ignored** — nothing here is committed
- All plans go here during planning mode — **not** in `~/.claude/plans/`
- **Naming**: `stageN_stepM_somename.md` (e.g. `stage1_step1_backend_setup.md`)
- **When done**: rename to `stageN_stepM_somename.done.md` for retrospective
- Move finalized docs to `architecture/` when approved
